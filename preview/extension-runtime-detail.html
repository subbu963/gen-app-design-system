Atoms introduced
  • Runtime chip. Same vocabulary as the kind chip — 9.5px mono pill — but distinguishes worker (gray) from deno (green). Pair with the kind chip on every row.
  • Process status dot. 7px circle with optional pulse. Green = running, gray = idle, faded = stopped, red = crashed/backoff. Re-used across the Logs viewer and Developer panel.
  • Host pill. Mono 10.5px pill on neutral background — the manifest's permissions.network entries the proxy enforces. Never editable here; this is a read-out.
  • Crashed-state copy is honest. "Backing off · 3 crashes · next attempt in 8s · disabled after 5." Numbers come from ext_runtime.rs — match them exactly.
  • iOS rationale block. Same three-part pattern as a denied-auth row — icon + headline + plain-language explanation + alternatives. Never just "unavailable".
