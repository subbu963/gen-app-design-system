Pattern notes
  • Status row. Three colors map to three states — green ("authorized"), orange ("in progress"), red ("failed"). Always paired with a 14px Lucide shield variant and the issuer URL on a second line.
  • Env-var bridge. stdio + OAuth is the awkward case — the wizard makes it explicit by labelling the field and showing the process.env.NAME call site in the footnote.
  • Re-authorize, not Refresh. Tokens refresh silently in mcp_auth.rs; the button is only for re-consent.
  • Loopback redirect is metadata, not configuration. Show the URI; never let the user edit it.
