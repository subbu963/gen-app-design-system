Pattern notes
  • Source chips are toggleable, not radio. Click any to subtract it. The all chip is a master + count, always first.
  • Off-state is opacity, not greyscale. Soft 0.5 — still scannable, still mono.
  • Severity colors stay row-level. Left border + 4% tint background only on warn/err; info rows are clean.
  • Export is one button, not a menu. If a user wants to share logs, the zip is the artifact — no per-format options.
