Pattern notes
  • Transport segmented stays at the top. Switching it resets every downstream field — we don't try to translate paths to URLs.
  • Probe is HTTP-only. stdio gets no probe — the npm package is its own contract; the user fills env names manually.
  • Auth-status row. The green "Authorized with …" panel is a reusable atom — also use it post-OAuth on Provider auth, and on per-secret detail.
  • Install is disabled until composeAuth() returns non-null. The footer button stays in the layout but goes opacity 0.4.
  • Terminal state replaces the body, not the chrome. Same sheet, same title — only the form swaps for the synthesized-providers list.
