Pattern notes
  • Quant chip is a typed badge. q4/q5/q6/q8/f16 map to green/blue/orange/purple/gray — color signals size class at a glance. Always 10px mono, no icon.
  • Download row is the canonical long-running atom. Filename + quant + percent + Cancel on row 1; thin 4px bar; mono "current / total · rate · ETA" on row 3. Reuse for MCP npx warm-up, canvas zip imports, anything streamed.
  • Active is a row state, not a sub-page. A check-circle in the accent tint and a small "active" pill — the model stays in place.
  • Empty hero uses the dashed-border + soft icon pattern. Same shape as the empty canvas — keeps "nothing here yet" voice consistent.
