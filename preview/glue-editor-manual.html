Pattern notes
  • Same chrome as the AI-authored mode. Breadcrumb, meta header, tab strip — all identical. The only swap is the right pane (chat dock → live preview) and the left's "AI is editing" → "Unsaved changes" status.
  • Props bar above the preview. Lets the user feed sample inputs without touching the source. Inputs match the widget's declared props (mono font, narrow editor).
  • Re-render on save, not on keystroke. A 128ms render means we don't need debounced live-recompile — the explicit ⌘S beat is the rhythm.
  • "Ask AI" is the gateway between modes. Toggling it slides the chat dock in from the right and demotes the preview to a tab — the two views are siblings, not competitors.
