# Iconography mapping — Lucide → SF Symbols

This system uses **Lucide** (web-embeddable) as a stand-in for **SF Symbols** (the native Apple icon set used in the actual Tauri app). When porting back to the real codebase, swap each Lucide name for its SF Symbols equivalent.

| Lucide | SF Symbols | Used in |
| --- | --- | --- |
| `message-circle` | `message` | Chat dock toggle |
| `plus` | `plus` | "Add widget" CTA, FAB |
| `settings` | `gearshape` | Settings entry |
| `cpu` | `cpu` | Providers (Settings → Brains) |
| `network` | `network` | Embeddings (Settings → Senses) |
| `puzzle` | `puzzlepiece.extension` | Extensions (Settings → Powers) |
| `file-text` | `doc.text` | Logs (Settings → Diaries) |
| `tag` | `tag` | Symbol on a widget |
| `check` | `checkmark` | Selected state |
| `grip-vertical` | `line.3.horizontal` | Widget drag handle |
| `x` | `xmark` | Close / dismiss |
| `rotate-cw` | `arrow.clockwise` | Retry |
| `key` | `key` | API key field icon |
| `sparkles` | `sparkles` | LLM-generated marker, "magic" moments |
| `cloud-sun` | *(widget content)* | Weather widget |
| `trending-up` | *(widget content)* | Stock widget |
| `calendar` | `calendar` | Calendar widget |
| `chevron-right` | `chevron.right` | List row affordance |
| `chevron-down` | `chevron.down` | Dock collapse |
| `arrow-up` | `arrow.up` | Chat send |
